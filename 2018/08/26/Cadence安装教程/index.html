Cadence安装教程

发表于 2018-08-26 | 分类于 Circuit Diagram |

简介

  介绍一款使用过的电路图设计软件Cadence。Cadence是一款专业的PCB设计和仿真软件，目前Cadence 16.6兼容Win8系统，特此讲解一下Cadence软件的安装使用。

软件安装

  1、安装包下载[Cadence]

  2、下载并解压文件，详细安装包文件内容如下

  3、双击Cadence-Allegro文件夹中的“setup.exe”开始正式的安装，先选择license manager进行安装。

  4、以D:\Cadence为安装路径，首先在路径下面建立三个文件License Manager、SPB_16.6、SPB_data，安装的时候分别选择相应的路径。License Manager安装到对应路径下的D:\Cadence\LicenseManager，然后一路Next。安装的过程中会跳出license选择，点击“cancel”。

  5、点击Product Installation安装对应的内容到
SPB_16.6/SPB_data路径下，然后一路Next，最后Finish，安装时间有点长，可以小憩一下。

软件破解

  6、以上步骤是完成软件的安装了，接下来就开始破解软件。
先把D:\Cadence\SPB_16.6\tools\pspice文件中的orSimSetup.dll复制到电脑桌面上，以便于后续的破解操作。

  7、解压”Cadence_SPB16.60_Crack”,生成“aspirin”文件夹，里面有三个文件License Manager/LicGen/SPB_16.6都需要使用。将”aspirin\Cadence\License Manager”下所有的文件（pubkey、pubkey1.30.exe 和lLicenseManagerPubkey.bat）复制到刚安装的Cadence License Manager目录下。

  8、复制完毕后，在“D:\Cadence\LicenseManager”下鼠标双击“License Manager”脚本程序,出现一个控制台窗口一闪而过，LicenseManager破解完成。

  9、按照上面步骤，还需要复制对应的破解文件到对应的主体文件路径下，将“aspirin\Cadence\SPB_16.6\tools”下所有的文件复制到刚安装的路径中“D:\Cadence\SPB_16.6\tools”。

  10、复制完毕后，在“D:\Cadence\SPB_16.6\tools”下鼠标双击“Tools”脚本程序，出现一个控制台窗口，等待破解程序运行，至此主体程序破解完毕。

  11、接下来生成License文件，进入“aspirin\Cadence\LicGen”文件夹，打开License模板文件“src.lic”,保存对src.lic文件的修改，然后双击“LicGen”命令脚本，重新生成License文件，将新生成的“license.lic”文件复制到“D:\Cadence”路径下备用。

  12、导入License文件，如果系统为64位的WIN7,需要先替换一个关键的文件，否则导入License文件会失败，解决的办法是将解压“cdslmd“文件得到”cdslmd.exe”复制到“D:\Cadence\LicenseManager“目录下，替换掉原有的文件。

  13、开始添加License文件路径，在“开始”菜单中运行“License Server Configuration Utility”，然后一路Next，Finish。



  14、运行License Clien Configurati on Utility，打开在空白处填上“5280@主机名”，点“next”，直至完成，5280是一个端口不能修改。

环境配置

  15、到这步Cadence Allegro 16.6破解基本上已经完成了，但还需要设置电脑环境变量，鼠标右键点击计算机，然后点击属性——>高级系统设置——>环境变量。先选择“新建”用户变量，新建如下变量：

  变量名：CDS_LIC_FILE

  变量值：5280@你的电脑名

  16、运行LMtools。选择ConfigSerices，第2个路径（path to license file）单击Browse添加证书，将License.bat换成我们生成的License.lic，然后点击save service。再点击start/stop/reread栏，先点Stop Server，在点击Start Server。

  17、把安装目录下的 D:SPB_16.6/tools/pspice 目录下剪切出来的 orsimsetup.dll放回原路径,（如果不用仿真部分删掉也无所谓），至此软件破解完成，可以使用。
