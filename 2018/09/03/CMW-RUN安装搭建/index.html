CMW RUN安装搭建

发表于 2018-09-03 | 分类于 RF Test Tool |

简介

   CMWRUN是一款配套CMW500使用的软件，通过这软件可实现射频的自动化测试，即使对手动操作不涉了解，只要对射频的测试项目有一定的认识，也能够较快的完成射频的相关测试。虽然厂家提供的原始脚本可以减轻我们的测试压力，但是却不能够满足我们更进一步的要求，比如说2G的脚本较为零乱，3G，4G没有RX的测试，部分项目不能统一测试等，所以需要对其进行优化以满足对自动化的要求。

软件安装

  1、安装Microsoft.NET和RSCMWrunSetup-1.8.5两个软件安装包（先安装Microsoft.NET再安装RSCMWrunSetup-1.8.5）。安装包下载[Microsoft.NET] [RSCMWrunSetup]
  

设备搭建

  2、配套设备：CMW500，屏蔽箱(带耦合板)或三角锥，射频连接线，电脑(安装CMWRUN)，网线等。
  

  3、建立CMWRUN与CMW500的连接

  3.1、电脑与CMW500用网线连接后。按setup键查看CMW500的IP地址，然后将电脑的IP地址修改成与CMW500-IP地址同一个网段但不能相同。例如CMW500-IP地址为172.22.1.5，可将电脑修改成172.22.1.6。
  

  3.2、CMW500 IP查看如下

  按“SETUP”按键，选择”System”栏目，在对应的Lan Services中：

  • Lan Remote: 对应于仪表后面板的网口，网口写有Lan Remote；
  • Lan Front: 对应于仪表前面板的网口；
  • 注：这两个网口可以用于控制仪表，其他网口的地址请勿更改。

  

  3.3、打开CMWRUN软件，设置其与仪表的连接

  点击菜单: “Resources->SCPI Connections…”，如下图

  

  4、添加设备

  第一次连接仪器时需添加新的设备，点击ADD如图：Alias部分输入CMW的别名，Resource Name部分输入CMW500的地址(格式必须为TCPIP0::172.22.1.4::INSTR，IP地址为CWM500的IP，如图所示)。

  

  点OK添加完毕后点击Test-Connection测试是否能够和CMW500连接上，下图为连接成功对话框。

  

  5、选择License Keys

  连接成功之后选择CMWRUN使用时需要的License Keys来源，点击菜单“Options->License Keys->Remote Device…”，如下图所示:

  

  弹出“Remote License Server”界面，在编辑框中填入CMW的“Resource Name”（和上一步的Resource Name一致），然后点击“Access!”查询CMW中支持License的信息，如下图所示: 添加后点OK。

  

  成功添加License Keys后，软件会提示重启CMWRUN软件。关闭CMWRUN软件，重新打开软件后会自动控制CMW500，这样就可以进行自动化测试。
